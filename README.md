# SOPT-SERVER-nodejs


* 기술스택
   * 프로그래밍 언어 : JavaScript
   * 백엔드 개발 프레임워크 : Express
   * 데이터베이스 : MySQL

* 개발툴
   * IDE : Visual Studio Code
   * DB 구축/운영 : MySQL Workbench
   * API 테스트 : Postman

* 배포
   * AWS EC2 - 클라우드 환경 컴퓨팅 시스템
   * AWS RDS - 클라우드 환경 데이터베이스 관리 시스템
   * AWS S3 - 클라우드 환경 객체 스토리지
